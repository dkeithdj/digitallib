
# Digital Library

> #### Dependencies in order to work:
> 
> - MySQL
> - maven

**NOTE:** Make sure these dependencies are on your system path

> #### To run the project:
>
> do this inside your command prompt and go to the directory where *pom.xml* resides.
> `mvn assembly:single`
> `java -jar target\[*name of .jar file*]`
